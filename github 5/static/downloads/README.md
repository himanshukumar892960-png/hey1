# GlobleXGPT App Downloads

This directory contains the downloadable app files for different platforms.

## Add Your App Files Here

Place your compiled application files in this directory with the following names:

- **Android**: `GlobleXGPT.apk`
- **Windows**: `GlobleXGPT-Setup.exe`
- **macOS**: `GlobleXGPT.dmg`
- **Linux**: `GlobleXGPT.AppImage`

## File Size Recommendations

- Keep files under 100MB for better download experience
- For larger files, consider using a CDN or cloud storage
- Compress files when possible

## Important Notes

1. **Security**: Ensure all files are scanned for malware before uploading
2. **Versioning**: Consider adding version numbers to filenames
3. **Updates**: Replace files when new versions are available
4. **Testing**: Test downloads on different devices before deployment
